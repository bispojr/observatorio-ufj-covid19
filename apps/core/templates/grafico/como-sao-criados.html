Como são criados os gráficos?

Todos os gráficos são criados a partir de dados oficiais publicados pelas secretarias de saúde de cada município. Estes dados oficiais são emitidos normalmente por meio de documentos formais chamados de boletins epidemiológicos. Alguns municípios preferem publicar os seus dados oficiais por meio de vídeos ou posts nas redes sociais.

Diariamente, um membro de nossa equipe monitora a atualização destes dados oficiais. Sempre que possível, este dado é atualizado imediatamente após a sua publicação. A nossa base de dados automaticamente atualiza o gráfico na página, de forma que o gráfico que você visualiza são com as informações mais atualizadas possíveis. Em cada página, é possível você saber o dia da última atualização do gráfico que você está visualizando.
